These are the scripts for the ZEB CHINA project and two scripts for the spatial analysis of the SWISSSTES project

Important scripts

preprocess_buildings_height.py  : This script generates the required input files for generate_blocks.py and assignes building heights to 
generate_blocky.py              : This script allows to write out the library of superblocks (with buildings etc.) and create CESAR-P 
stess_codes.py                  : Script to download and polygonize all Amtliche Vermessung polygons
stess_identify_plots.py         : Script to filter/process all individual polygons to identify suitable footprints. This script needs to considerably expaned to integrate additional data and criteria.
